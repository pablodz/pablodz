## Código con resaltado de sintaxis

Los bloques de código usan Shiki con el tema *Vesper* (naranja sobre negro,
como debe ser). Soporta cualquier lenguaje:

```js
// El clásico: debounce sin dependencias
export function debounce(fn, delay = 300) {
  let timer = null;
  return (...args) => {
    clearTimeout(timer);
    timer = setTimeout(() => fn(...args), delay);
  };
}

const onResize = debounce(() => {
  console.log(`viewport: ${window.innerWidth}×${window.innerHeight}`);
}, 150);

window.addEventListener('resize', onResize);
```

```go
// También Go, claro
func worker(ctx context.Context, jobs <-chan Job, results chan<- Result) {
	for {
		select {
		case <-ctx.Done():
			return
		case job := <-jobs:
			results <- process(job)
		}
	}
}
```

El código inline también funciona: `const nodes = ['alpha', 'beta', 'gamma']`.
